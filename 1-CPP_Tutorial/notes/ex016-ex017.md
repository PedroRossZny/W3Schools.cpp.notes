# C++ Constants

## Exercise 016

When we do not want the value of a variable to be changed, we use the `const` keyword,
which declares the variable as a *constant* — meaning it is *immutable and read-only*. You
should declare a variable as a constant when it is likely to hold a value that never
changes, as shown in **ex016**.

## Exercise 017

When declaring a variable as a constant, we must always assign a value to it at the time
of declaration. If we do it separately, an error will occur, since we cannot change the
value of a constant after it has been declared, as shown in **ex017**.
