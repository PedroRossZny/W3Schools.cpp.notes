# C++ Operators

## Exercise 030

Operators are used to perform operations on variables and values, such as the `+` operator
which adds two values together. In addition to adding two values, `+` can also be used to
add a variable and a value, or two variables together.

Operators are divided into the following groups:

- Arithmetic operators
- Assignment operators
- Comparison operators
- Logical operators
- Bitwise operators
