# C++ Declare Multiple Variables

## Exercise 013

To declare multiple variables of the same type, we can declare them all on a single line,
separating each name and its assigned value with a comma, as shown in **ex013**.

---

## Exercise 014

We can also assign the same value to multiple variables on a single line. First we declare
the variables with the same type, then on another line we chain them together, each
followed by an *equal sign*, with the value assigned at the end. This way all of them will
hold the same value, as shown in **ex014**.
