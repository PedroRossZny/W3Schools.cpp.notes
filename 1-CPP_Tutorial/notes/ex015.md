# C++ Identifiers

## Exercise 015

All variables must be identified with *unique names* called *identifiers*. These can be
short names or more descriptive ones — the latter being the most recommended, as it
provides better understanding and easier maintenance. The general rules are as follows:

- Names can contain letters, digits, and underscores.
- Names must begin with a letter or an underscore (`_`).
- Names are case-sensitive — a variable like `myVar` is different from `myvar`.
- Names cannot contain whitespace or special characters such as `!`, `#`, `%`, etc.
- Reserved words of the language, such as `int`, cannot be used as names.
