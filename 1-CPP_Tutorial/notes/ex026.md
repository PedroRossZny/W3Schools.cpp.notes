# C++ String Types

## Exercise 026

The `string` type is used to store a sequence of characters, in other words, text. In
basic usage it behaves like a native type, even though it is not one. String values are
delimited by double quotes. To use strings, we need to include an additional header file,
the `<string>` library, or alternatively just include `<iostream>`, which also covers it,
as shown in **ex026**.
