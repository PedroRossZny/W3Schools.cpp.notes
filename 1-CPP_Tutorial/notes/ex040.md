# C++ Logical Operators

## Exercise 040

Logical operators can also test boolean values, just like comparison operators. They are
used to determine the logic between variables or values:

| Operator | Name | Description | Example |
|----------|------|-------------|---------|
| `&&` | Logical AND | Returns true if both statements are true | `x < 5 && x < 10` |
| `\|\|` | Logical OR | Returns true if at least one statement is true | `x < 5 \|\| x < 4` |
| `!` | Logical NOT | Inverts the result, returns false if the result is true | `!(x < 5 && x < 10)` |
