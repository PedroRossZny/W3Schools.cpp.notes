# C++ Introduction

## Exercise 001

To run C++ on your own computer, you will need two things: a text editor and a compiler,
and there are many options to choose from.

An IDE (Integrated Development Environment) is used to both edit and compile code.
Among the most popular IDEs are *Code::Blocks*, *Eclipse*, and *Visual Studio*.

We will be using Visual Studio — you can download it at this [link](https://visualstudio.microsoft.com/downloads/).

To create our first project:
1. Go to New Project → Empty Project
2. Give the project a name
3. Click Create
4. Open the Solution Explorer
5. Under Source Files, create a new `.cpp` file

After that, we can write our first piece of code. To run it, press *CTRL + F5* (compiles
and runs without the debugger) or just *F5* (compiles and runs with the debugger).

Congratulations — you have run your first C++ program!
The example code can be found in **ex001**.
