# C++ Assignment Operators

## Exercise 035

Assignment operators are used to assign values to variables. The **addition assignment
operator** `+=` adds a value to a variable. The full list of assignment operators is:

| Operator | Example | Equivalent to |
|----------|---------|---------------|
| `=` | `x = 5` | `x = 5` |
| `+=` | `x += 3` | `x = x + 3` |
| `-=` | `x -= 3` | `x = x - 3` |
| `*=` | `x *= 3` | `x = x * 3` |
| `/=` | `x /= 3` | `x = x / 3` |
| `%=` | `x %= 3` | `x = x % 3` |
| `&=` | `x &= 3` | `x = x & 3` |
| `\|=` | `x \|= 3` | `x = x \| 3` |
| `^=` | `x ^= 3` | `x = x ^ 3` |
| `>>=` | `x >>= 3` | `x = x >> 3` |
| `<<=` | `x <<= 3` | `x = x << 3` |

As shown in **ex035**.

## Exercise 036

Compound assignment operators are a more concise way of writing operations where you use
the same variable on both sides of the assignment. Instead of writing `x = x + 5;` for
example, we can write `x += 5;`. These operators make the code shorter and easier to read.
They are called *compound assignment operators* because they combine a regular operator,
such as `+`, `-`, or `*`, with the assignment operator `=` into a single operator, as
shown in **ex036**.
