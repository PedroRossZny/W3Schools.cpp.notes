# C++ New Lines

## Exercise 006

To insert a new line in the output, we use the `\n` character. We can also add another
`<<` operator and place the `\n` character after the text. Two `\n` characters in a row
create a blank line, as shown in **ex006**.

## Exercise 007

Another way to insert a new line is with the `endl` manipulator. Both `\n` and `endl` are

## Additional

The `\n` character is called an *escape sequence*. It forces the cursor to move to the
beginning of the next line on the screen. Other examples of valid escape sequences are:

- `\t` — creates a horizontal tab (like pressing the Tab key)
- `\\` — inserts a backslash character
- `\"` — inserts a double quote character
