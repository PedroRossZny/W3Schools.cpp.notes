# C++ User Input

## Exercise 020

`cout` is used to display values. Now we will use `cin` to receive them. `cin` is a
predefined variable that reads data from the keyboard using the extraction operator `>>`.
In a real example, the user can enter a value, and `cin` makes that input possible —
followed by the extraction operator `>>` and then the variable that will receive the value
typed by the user.

- `cout` is pronounced *see-out*, represents *output*, and uses the insertion operator `<<`.
- `cin` is pronounced *see-in*, represents *input*, and uses the extraction operator `>>`.
