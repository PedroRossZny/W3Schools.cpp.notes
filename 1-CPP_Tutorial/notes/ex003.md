# C++ Statements

## Exercise 003

A program is a list of instructions to be executed by a computer. In a programming
language, these programming instructions are called statements.

The statement in **ex003** instructs the compiler to print the texts "Hello World!" and
"Have a good day!" on the screen. It is very important that you end each statement with a
semicolon, otherwise errors will occur and the program will not run.

When we have many statements — which is the case for most C++ programs — they are executed
one by one, in the order they are written. In **ex003** we have three statements:

1. `cout << "Hello World!";`
2. `cout << "Have a good day!";`
3. `return 0;`

The first statement is executed first, followed by the second, and finally the third,
which ends the program.
