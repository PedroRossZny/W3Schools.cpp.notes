# C++ Character Data Types

## Exercise 025

The `char` type is used to store a single character, which must be enclosed in single
quotes, such as `'A'` or `'c'`. Another option is to use ASCII values to display specific
characters. A full list of ASCII values can be found at the
[ASCII Reference Table](https://www.w3schools.com/charsets/ref_html_ascii.asp), as shown
in **ex025**.
