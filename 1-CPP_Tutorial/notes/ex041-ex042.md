# C++ Operator Precedence

## Exercise 041

When a calculation contains more than one operator, C++ follows the rules of **order of
operations** to decide which part to calculate first. For example, multiplication happens
before addition. If you want addition to happen before multiplication, parentheses must be
used. Always use them when you want to guarantee that the calculation is performed in the
expected order, as it also makes the code easier to read.

Some common operators listed from highest to lowest precedence:

| Operator | Description |
|----------|-------------|
| `()` | Parentheses |
| `*` `/` `%` | Multiplication, division, modulus |
| `+` `-` | Addition, subtraction |
| `>` `<` `>=` `<=` | Comparison |
| `==` `!=` | Equality |
| `&&` | Logical AND |
| `\|\|` | Logical OR |
| `=` | Assignment |
